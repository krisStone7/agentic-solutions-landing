# Agentic Solutions Landing Page

Minimal Vite-style React starter for a one-page business landing page for Agentic Solutions in Los Angeles, CA.

## Run locally

```bash
npm install
npm run dev
```

## Included files

- `package.json`
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles.css`

The page includes sections for hero, services, how it works, outcomes, who it's for, and a contact CTA.
